### `01` &ensp; Toolbox — POS & Inventory

![](https://img.shields.io/badge/Production-Live-2d6a4f?style=flat-square&labelColor=0d1117) &ensp; ![](https://img.shields.io/badge/Users-60%2B-C9A84C?style=flat-square&labelColor=0d1117) &ensp; ![](https://img.shields.io/badge/SKUs-1%2C000%2B-8892a4?style=flat-square&labelColor=0d1117)

Enterprise POS managing a full product catalog across a distributed team — built offline-first, synchronized in real-time.

**Architecture highlights**
- Offline-first PWA with Service Worker + IndexedDB caching for catalog performance at scale
- Socket.IO multi-session stock sync — 60+ concurrent users, zero conflicts
- Barcode scanning with live inventory validation
- Advanced cart engine: persistence, session recovery, abandoned-cart detection
- Role-based auth, Zod validation, XSS/CORS/rate-limit hardening
- Full audit trail with CSV / XLSX / JSON export

### `03` &ensp; Finance Payroll Dept. — Financial System

![](https://img.shields.io/badge/Status-Deployed-2d6a4f?style=flat-square&labelColor=0d1117) &ensp; ![](https://img.shields.io/badge/Year-2026-C9A84C?style=flat-square&labelColor=0d1117)

Enterprise financial platform covering the full money lifecycle — vouchers, invoicing, payroll, loans, and multi-level approvals.

**Architecture highlights**
- State-machine approval flow: Draft → Submitted → Approved → Released
- Three voucher types: Cash · Check · Petty Cash
- Sales invoicing with PDF generation and customer database
- Payroll engine with multi-period support and bulk processing
- Employee loans system — 4 loan types, color-coded status tracking
- Dynamic dashboards with pie / bar / area charts and KPI widgets
- Multi-sheet Excel exports with professional styling
