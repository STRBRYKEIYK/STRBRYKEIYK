### `01` &ensp; Toolbox — POS & Inventory System

![](https://img.shields.io/badge/Production-Live-1a7a3c?style=flat-square&labelColor=0d1117)&ensp;![](https://img.shields.io/badge/Users-60%2B-4ade80?style=flat-square&labelColor=0d1117)&ensp;![](https://img.shields.io/badge/SKUs-1%2C000%2B-6aab7e?style=flat-square&labelColor=0d1117)

> Full-stack web-based POS and inventory management system designed for enterprise retail operations — built offline-first, synchronized in real-time across all sessions.

**Key Achievements**
- Offline-first PWA with Service Worker + IndexedDB caching; full functionality without internet, auto-sync on restore
- Socket.IO multi-session real-time stock sync across 60+ concurrent users with zero conflicts
- Barcode scanning with live inventory validation and automated low-stock alerting
- Advanced cart engine: session persistence, history recovery, and abandoned-cart detection
- Supported 1,000+ SKUs with search, filtering, grid/list views, and pagination
- Role-based access control, Zod schema validation, XSS/CORS/rate-limit hardening
- Comprehensive employee audit trail with transaction logging and performance metrics
- Export pipeline: CSV, XLSX, JSON for analytics and reporting

**Tech Used**

### `02` &ensp; Procurement Dept. — Supply Chain Hub

![](https://img.shields.io/badge/Status-Active-1a7a3c?style=flat-square&labelColor=0d1117)&ensp;![](https://img.shields.io/badge/Scope-Full_Lifecycle-4ade80?style=flat-square&labelColor=0d1117)

**Key Achievements**
- Real-time inventory dashboard with KPI widgets, stock-level alerts, and quick-action interface

- Supplier database with performance tracking and full communication history
- Comprehensive employee activity audit trail with real-time WebSocket order-status notifications
- Responsive UI with full mobile and desktop parity

### `03` &ensp; Finance Payroll Dept. — Financial System

![](https://img.shields.io/badge/Status-Deployed-1a7a3c?style=flat-square&labelColor=0d1117)&ensp;![](https://img.shields.io/badge/Year-2026-4ade80?style=flat-square&labelColor=0d1117)

> Enterprise-grade financial platform covering the full money lifecycle — vouchers, invoicing, payroll, loans, expense tracking, and multi-level approval workflows.

**Key Achievements**
- State-machine approval engine: Draft → Submitted → Approved → Released across all financial documents
- Three voucher types (Cash · Check · Petty Cash) with complete workflow management
- Sales invoicing system: line item management, PDF generation, and customer database integration
- Payroll operations: employee master data, multi-period support, and bulk batch processing
- Employee loans system with 4 loan types, color-coded status tracking, and disbursement management
- Expense tracking with account classification and budget target monitoring
- Monthly bills management with vendor database and recurring billing cycle tracking
- Dynamic dashboards — pie / bar / area charts, KPI widgets, and trend analytics
- Multi-sheet Excel exports with professional styling and date-range filtering

**Tech Used**
